# Descripción
Unzip this archive and find the flag.
# Solución
Descomprimir el .zip y utilizar el grep con -rni para poder obtener la flag

- Utilizar el webshell integrado, después descargar el .zip y descomprimirlo, luego utilizar el comando grep con -rni "pico", para que con ello se haga la búsqueda de la flag

picoCTF{gr3p_15_m4g1c_ef8790dc}

- Puede además utilizarse alguna distro de linux para hacerlo localmente
