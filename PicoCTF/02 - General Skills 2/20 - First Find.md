# Descripción
Unzip this archive and find the file named 'uber-secret.txt'
# Solución
Descomprimir el zip y usar el grep con -rni para obtener la flag

- Utilizar el webshell integrado, después descargar el .zip y descomprimirlo, luego utilizar el comando grep con -rni "picoCTF", para que con ello se haga la búsqueda de la flag

picoCTF{f1nd_15_f457_ab443fd1}

# Notas adicionales
- Puede además utilizarse alguna distro de linux para hacerlo localmente
